Maximum number of sets with restricted intersection number
For fixed $k$ and $t$, as $n\to\infty$, the maximum number of $k$ elements subsets of $[n]$ that any two sets intersect in at most $t-1$ elements is roughly \[\binom{n}{t}/\binom{k}{t}\] See *Erdos-Hanani conjecture* and *Rodl Nibble Method*.

k-average-free set
[Behrend,46] there is a k-average free set $S\subseteq[u]$ of size $n$ with \[u\le k^{O(1/\epsilon)}n^{1+\epsilon}\] for any $\epsilon>0$.
