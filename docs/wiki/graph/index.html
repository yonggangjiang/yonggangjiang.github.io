Graph Theory

Reference: A First Course in Graph Theory, or this.

Blocks

A vertex-cut of $G$ whose removing will lead to $G$ being inconneted.

A non-separable graph do not have a vertex-cut.

A maximal nonseparable subgraph is a block of the graph.

Another Definition
Define relationship $\sim$ on all edges of $G$, where $a\sim b$ iff. edges $a$ and $b$ lies on a single circle. This is an equivalent relaitionship. Each equivalent class is a block. Any block is a equivalent class.
Properties of Blocks
Any two blocks have at most $1$ vertex in common, which is a vertex-cut.

Connectivity

$\kappa(G)$ is the least number of vertices removing which could disconnect $G$, or to let $G$ be a trivial graph. Obviously, $0\le\kappa(G)\le n-1$. $G$ is $k$-connected if we need to remove at least $k$ vertices to make it disconnected or be trivial, which means $\kappa(G)\ge k$.

$\lambda(G)$ is the least number of edges removing which could disconnect $G$ for $G\not=K_1$. While we define $\lambda(K_1)=0$. We have $0\le\kappa(G)\le n-1$ since we can use at most $n-1$ edges to disconnect one vertex. SImilarly, $G$ is $k$-edge-connected if $\lambda(G)\ge k$.

Let $\delta(G)$ be the minimum degree among vertices in $G$. Then we have

\[\kappa(G)\le\lambda(G)\le\delta(G)\]

Here are some theorems regarding connectivity.

Theorem 1
If every vertices of graph $G$ has degree $3$ (this is defined as cubic graph), then $\kappa(G)=\lambda(G)$.
Theorem 2
For any graph with $m$ edges and $n$ vertices we have $\kappa(G)\le\lambda(G)\le\lfloor\frac{m}{2n}\rfloor$.
This bound is tight: The Harary Graph which is a $r$-regular graph has vertex conectivity $r$.
Theorem 3
If $G$ is connected with $n\ge 3$, then $G^2$ is $2-connected$.
The following theorem is proved by induction on the size of graph. It holds for each-connectivity (max-flow=min-cut).

Menger's Theorem
For nonadjacent vertices $u,v\in V$, the minimum number of vertex whose removing will disconnected $u$ and $v$, equals to the maximum number of $u-v$ path that are vertex disjoint besides $u,v$.
Theorem 4
If $G$ is $k$-connected, then every $k$ nodes in $G$ lies on a common circle.
