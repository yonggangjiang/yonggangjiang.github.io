Pseudoinverse and Projection

Pseudoinverse
For $A\in\mathbb{R}^{m\times n}$, $A+\in\mathbb{R}^{n\times m}$ is a pseudoinverse of $A$ if the following criteria are satisfied.
1. $AA^+A=A$.
2. $A^+AA^+=A^+$.
3. Both $AA^+$ and $A^+A$ are symmetric.
Suppose we want to solve $Ax=b$. If there is ganranteed to be a solution $x$, then we have $AA^+b=AA^+Ax=Ax=b$, which means $A^+b$ is a solution. This only need the first condition.

Suppose $A$ has full rank, then the pseudoinverse is simply $(A^TA)^{-1}A^T$ or $A^T(A^TA)^{-1}$ depending on the size of $n,m$.

Projection
Suppose $A$ is combine by $n$ basises in $\mathbb{R}^m$ (column vector, which means $A\in\mathbb{R}^{m\times n}$), those vector span a subspace $V$. Then for any $b\in\mathbb{R}^m$, the projection from $b$ to the space can be represented as $P\cdot b$ where $P=A(A^TA)^{-1}A^T$, which is a linear function on $b$. Note that $P^k=P$ for any $k\in\mathbb{N}^+$.

Real Symmetric Matrix

Suppose $A\in\mathbb{R}^{n\times n}$ is a real symmetrix matrix.

Eigenvalues and Eigenvector

According to foundamental theorem of algebra, any matrix has $n$ complex eigenvalues.

For any two eigenvalue $\lambda_1,\lambda_2$ of $A$, suppose $v_1,v_2$ are the corresponding eigenvectors, then we have

\[\bar{v_1^T}Av_2=\bar{v_1^T}\bar{A^T}v_2=\bar{v_1^T}\lambda_2v_2=\bar{v_1^T}\bar{\lambda_1}v_2\]
\[\]

Thus, when $v_1=v_2=v$, we get $\lambda_1=\bar{\lambda_1}=\lambda$, which means any eigenvalue of the matrix is a real number. That means $|A-\lambda I|=0$, and there exists a real eigenvector corresponding to $\lambda$.

When $\lambda_1\not=\lambda_2$, we get $\bar{v_1^T}v_2=0$. That means any two real eigenvectors corresponding to difference eigenvalues are orthogonal. Note that $v_1$ can be complex vector, just consider $v_1+iv_1$ for the real vector $v_1$.

If $V$ is composed by the $n$ eigenvectors of $A$ (in each column), then $AV=VQ$ where $Q$ is a diagonal matrix with $n$ eigenvalues of $A$ on the diagonal. Since $V$ has full rank, we get $A=VQV^{-1}=VQV^T$.

Positive Semidefinite

If for any $x$ we have $x^TAx\ge 0$, then $A$ is called semidefinite. If $A$ has a negetive eigenvalue $\lambda$and the corresponding eigenvector is $v$, then $v^TAv=\lambda v^Tv<0$. Thus, all eigenvalues of $A$ are non-negetive.

Similarly, if for any $x$ we have $x^TAx>0$, then $A$ is called positive definite, and all its eigenvalues are positve.

Recall that $A=VQV^T$, by write $Q^{\frac{1}{2}}$ as the matrix with the squre root of eigenvalues of $A$ on diagonal, we get $A=V^TQ^{\frac{1}{2}}Q^{\frac{1}{2}}V=B^TB$ where $B=Q^{\frac{1}{2}}V$.

Eigenvalues and Singular Values

For any complext matrix $A$, the matrix $A^\dagger A$ is real symatric and positive semidefinite, thus, it has $n$ non-negative eigenvalues. The squre root of each of them is called Singular Value.

Min-Max theorem

Min-Max theorem for eigenvalues
Suppose $A$ is an $n\times n$ Hermitian matrix with eigenvalues $\lambda_1\le...\le \lambda_k\le....\le\lambda_n$, then
\[\lambda_k=\min_{U\subseteq \mathbb{R}^n,\dim(U)=k}\left(\max_{x\in U,x\not=0}\frac{x^\dagger Ax}{x^\dagger x}\right)\] \[\lambda_k=\max_{U\subseteq \mathbb{R}^n,\dim(U)=n-k+1}\left(\min_{x\in U,x\not=0}\frac{x^\dagger Ax}{x^\dagger x}\right)\]

We use the Min-Max theorem on $A^\dagger A$ for arbitrary $A$, the theorem becomes

Min-Max theorem for singular values
Suppose $A$ is an arbitrary matrix with sigular values $\sigma_1\le...\le \sigma_k\le....\le\sigma_n$, then
\[\sigma_k=\min_{U\subseteq \mathbb{R}^n,\dim(U)=k}\left(\max_{x\in U,x\not=0}\frac{||Ax||_2}{||x||_2}\right)\] \[\sigma_k=\max_{U\subseteq \mathbb{R}^n,\dim(U)=n-k+1}\left(\min_{x\in U,x\not=0}\frac{||Ax||_2}{||x||_2}\right)\]

Here $||x||_2$ means $x^\dagger x$. This theorem leads to the spectrum norm of a matrix, which is defined to be the largest singular value of the matrix, also measure how big can the matrix scale up a vector.

Relation between eigenvalues and singular values

Weyl’s inequality and Schur–Horn theorem.

For Normal Matrix (includes Hermitian matrix), we have the equality $|\lambda_k|=\sigma_k$ for any $k$. Here $|\cdot|$ is the size of the complex number.

Norm

General Norm
The function $||\cdot||$from $\mathbb{R}^n\to\mathbb{R}$ is a norm if the following conditions are satisfied for any $u,v\in\mathbb{R}^n$ and $c\in\mathbb{R}$.
1. $||c\cdot u||=|c|\cdot||u||$.
2. $||u+v||\le||u||+||v||$
3. $||u||=0\leftrightarrow u=0$

For a positive definite matrix $A$, $||u||_A:=\sqrt{u^TAu}$ can be proved as a norm.

Dual Norm
If $||\cdot||$ is a norm, then define the dual norm
\[||x||^*:=\sup_{||y||\le 1}\langle x,y\rangle\]
We have the generalized Cauchy-Schwarz inequality $\langle x,y\rangle\le||x||\cdot||y||^*$.

$\forall p,q\ge 1$ and $1/p+1/q=1$, $||\cdot||_q$ is a dual norm of $||\cdot||_p$. That is due to the Lagrange multipliers, we get

\[x=\lambda qy^{q-1}\]
Here $x$ represents for any entry of $x$, and we want $y$ to optimize $\langle x,y\rangle$, which must satisfy $\sum y^q=1$. That means $\sum xy=\lambda q$. And if we let $p=q/(q-1)$, we can also get
\[x^p=(\lambda q)^py^{q}\]
Which means $\lambda q=||x||_p$.

Matrices Associated to Graphs

Given a undirected graph $G=(V,E)$. Let $n=|V|$ and $m=|E|$.

Laplacian

Adjacency matrix $A$ is defined as: $A_{u,v}=1$ iff. $(u,v)\in E$, otherwise $A_{u,v}=0$.

Degree matrix $D$ is defined as a diagonal matrix, with $D_{u,u}=d_u$.

If we assign arbitrary direction to the edge in $G$, we can difine vertex-edge incidence matrix $B\in {-1,0,1}^{n\times m}$ while $B_{u,e}=1$ iff. $e=(\cdot, u)\in E$ and $B_{u,e}=1$ iff. $e=(u,\cdot)\in E$.

Graph Laplacian of G is defined as a symmetric matrix $L=D-A$. We can also verify that $L=BB^T$.

If $G$ has weight diagonal matrix $W$ where $W_{e,e}$ is the weight of $e$, then the Laplacian is defined as $BWB^T=D-A$, where $D$ and $A$ now also contain weight.

Rank

Notice that if we add all row of $L$, then we get zero vector: that means $L$ is has no full rank.

If $G$ is connected, then we can prove the rank of $L$ is $n$: For any vector $v$ that is not parallel to $1$, which means there exists an entry which is not the maximum value of the vector $x_i$. Now if $Lv=0$, then the vertices set that indexed as the maximum value of $v$ must be disjoint with the vertex indexed as the smallest value. To see this, notice that any vertex which put the maximum value on the diagonal can not sum to $0$ if there exists an edge connecting a smaller value; and the vertex with the smallest value definitely cannot sum to $0$ if there is an edge associate with a bigger value.
